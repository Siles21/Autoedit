Dein Video bekommt
automatisch Grafiken.

Du hast ein fertig geschnittenes Video. Autoedit erkennt die wichtigen Stellen — Zahlen, Aufzählungen, Aussagen — und baut dafür schicke Einblendungen, genau im richtigen Moment. Du musst es nur einmal einrichten.

Dein Schnitt
nur gesprochenes Wort
→
Nachher
Schnitt + Grafiken
Zahlen, Listen, Reveals

Du tippst keinen Code

Du kopierst beim Einrichten 3 Zeilen — verstehen musst du sie nicht.

Alles ist dabei

Ein Befehl installiert alles Nötige automatisch auf deinem Mac.

Du redest mit Claude

Statt Befehle zu lernen, sagst du einfach, was du willst.

Was du brauchst

Einmalig auf deinem Mac — die meisten haben das eh schon.

  • Ein Mac · macOS
  • Adobe Premiere Pro · nur wenn die Grafiken direkt rein sollen
  • Dein fertig geschnittenes Video · als Datei (z. B. final.mp4)

Einrichten — einmalig

Öffne das Programm „Terminal" (mit Cmd+Leertaste „Terminal" suchen). Dann diese drei Kästchen nacheinander hineinkopieren und jeweils Enter drücken.

1

Das Programm laden

Holt Autoedit auf deinen Computer.

git clone https://github.com/Siles21/Autoedit && cd Autoedit
2

Automatisch installieren

Richtet alles Nötige selbst ein. Das dauert ein paar Minuten — einfach laufen lassen.

./setup.sh
⏳ Beim ersten Mal lädt es viel herunter. Warten, bis „Setup done" erscheint.
3

Startklar machen

Aktiviert die Umgebung. Das machst du jedes Mal, wenn du ein neues Terminal-Fenster öffnest.

source .venv/bin/activate

Verwenden — einfach reden

Wenn du Claude Code nutzt, brauchst du keine Befehle. Du gibst Claude nur dein Video und sagst, was du willst.

1

Video finden

Such dein fertiges Video im Finder (z. B. final.mp4).

→
2

Pfad kopieren

Rechtsklick auf das Video, dann die ⌥ Option-Taste halten → „… als Pfadname kopieren".

→
3

In Claude einfügen

In Claude mit Cmd+V einfügen und deinen Wunsch dazuschreiben.

So sieht deine Nachricht an Claude aus:
/Users/du/Desktop/final.mp4 Mach mir Sub-Animationen für dieses Video — Zahlen und Aufzählungen einblenden.

Claude macht dann die ganze Arbeit und fragt dich vor dem Einbauen um Freigabe:

Du
„Bau mir Sub-Animationen für mein Video final.mp4 — Zahlen und Aufzählungen einblenden."
Claude
Ich höre das Video ab, finde die wichtigen Stellen und schlage dir einen Plan vor. Schau ihn dir an — passt alles?
Du
„Ja, sieht gut aus — bau's ein."
Claude
Erledigt. Die Grafiken liegen jetzt im Schnitt / als Datei zum Reinziehen bereit.

Einmal verbinden: Damit Claude den Skill kennt, kopierst du diese eine Zeile ins Terminal — danach versteht Claude einfach „mach Sub-Animationen".
ln -s "$(pwd)/skill" ~/.claude/Skills/premiere-sub-animations

Und dann?

Du prüfst die Grafiken in Premiere. Gefällt etwas nicht? Sag es Claude — „mach die Zahl größer", „verschieb das nach hinten" — und es wird angepasst. Kein Code, nur Sprache.
